Tipuri de date

Deschide consola pentru a vedea executia codului javascript.

In javascript avem urmatoarele tipuri de date:

  1. Number ( primitiv )
  2. *Float ( este tot number )
  3. Boolean ( primitiv )
  4. Undefined ( primitiv )
  5. String ( primitiv )
  6. Null ( primitiv )
  7. Array ( liste )
  8. Object

Indentificarea tipului de date din javascript

Pentru a indetifica un tip de data in javascript folosim operatorul typeof

Primitive vs complexe

Tipurile de date primitive, reprezinta tipurile de date simple, care au ca rol reprezentarea unei valori, si categorizarea/ insemnatatea ei.

Tipurile de date complexe, reprezinta structuri care pot contine mai multe tipuri de date primitive, dar chiar si tipuri de date complexe in interiorul lor. Nu sunt restrictionate la un anumit numar de informatii.

Number

Este un tip de data folosit pentru a reprezenta/recunoaste numerele, cu sau fara virgula, cele cu virugla se mai numesc si float.

Number este un tip de data primitiv.

Boolean

Boolean este un tip de data primitiv si el.

Reprezinta valori de adevarat sau fals ( true sau false ).

Undefined

Undefined este un tip de data primitiv.

Undefined indica o variabila nedefinita

String

String este un tip de data primitiv.

String ne indica ca avem o valoare text, adica un cuvant sau o propozitie.

Se mai poate spune ca stocheaza o serie de caractere.

Null

Null este un tip de data primitiv.

Null este folosit pentru indica ca o anumita variabila nu are valoare, adica are valoarea nimic.

Se mai poate spune ca este reprezentarea lui 'nimic' in javascript.

Array ( liste )

Array-urile sunt considerate tipuri de date complexe, pentru ca pot contine valori cu mai multe tipuri de date in interiorul lor.

Sunt folosite pentru a stoca mai multe valori/informatii sub forma de lista.

Listele folosesc index, pentru a indica pozitia unui element.

Indexul incepe tot timpul de la 0, adica elementul cu numarul 1 din lista, primul element va avea indexul 0.

Indexul ne ajuta sa accesam un element din lista noastra, folosind pozitia lui.

Obiectele

Sunt un tip de data complex in javascript, folosit pentru a stoca informatii strans legate intre ele .

Exemplul unei masini, care poate avea urmatoarele proprietati: culoare, an de fabricatie, tip carburant, tara de origine ...
