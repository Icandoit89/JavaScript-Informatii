Srtucturi decizionale

Structurile decizionale sunt modalitati prin care in javascript putem evalua rezultatul unei compartii/ executii de cod si bazat pe rezultat putem executa o parte de cod sau nu.

Structura If ( daca )

Ne ajuta sa evaluam o propozite logica, care returneaza adevarat/fals ( alb/negru, 1/0 ), si sa executam cod bazat pe valoarea ei.

Daca valoarea din interiorul parantezei este adevarata, codul din interiorul blocului IF va fi executat.

If poate fi folosit singur, fara alt bloc decizional

Structura Else If ( altfe daca )

Ne ajuta sa evaluam o propozitie deja evaluata de un IF, si sa decidem daca codul din interior va fi executat sau nu. Functioneaza la fel ca IF, daca propozitia este adevarata

Else if nu poate fi folosit decat dupa un block de IF

Structura Else ( altfel )

Daca propozitia evaluata in IF nu este adevarata, se va executa codul din interiorul blocului else, cel care urmeaza tot timpul dupa IF.

Acesta poate urma si dupa mai multe blocuri de else/if cu aceeasi regula, daca nu se va executa codul din una din ele, se va executa codul din else.

Swtich ( schimba )

Switch evalueaza o valoare simpla, compara cu "===".

Switch foloseste blocuri numite 'case :', in cazul in care valoarea este aceeasi cu cea specifica dupa cuvnatul case, codul din interiorul blocului case va fi executat

Are nevoie de break;, este un cuvant special folosit in switch pentru a intrerupe executia switch-ului in cazul in care cazul a fost intalnit si codul executat.

default, este cazul standard, necesar in toate switchurile, practic default este cazul folosit in situatia in care nici unul dintre cazurile definite nu este valid.
